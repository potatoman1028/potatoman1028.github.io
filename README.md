# 🥔🧪 POTATO'S LAB

POTATO'S LAB 기술 블로그에 오신 것을 환영합니다! 이 블로그는 Astro 프레임워크를 기반으로 구축되었으며, 저의 사이드 프로젝트 기록, AI 협업 경험, 그리고 다양한 기술적 탐구를 담는 공간입니다.

## ✨ 주요 기능
- **Astro 기반**: 빠르고 가벼운 정적 사이트 생성기(SSG).
- **AI 리포트**: AI 어시스턴트(Antigravity)와 함께 작업한 과정과 의사결정을 상세히 기록하는 전용 섹션 운영.
- **검색 시스템**: Pagefind를 이용한 빠른 내부 검색 지원.
- **댓글 시스템**: Giscus를 활용한 GitHub Discussions 연동.
- **SEO 최적화**: 구글 및 네이버 검색 노출을 위한 기술적 세팅 완료.

## 🛠️ 기술 스택
- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Content**: Markdown & MDX
- **Deployment**: GitHub Actions & GitHub Pages

## 🚀 로컬 개발 환경 실행
이 저장소를 로컬에서 실행하려면 다음 명령어를 사용하세요:

```bash
npm install
npm run dev
```
